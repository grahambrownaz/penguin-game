# 🐧 Pip the Penguin — Biome Quest

A self-contained 3D browser game built with [Three.js](https://threejs.org/).
Help Pip waddle across 9 worlds, rescue the eggs, and topple each world's boss!

## How to play

No install, no internet needed:

1. Download the project (green **Code → Download ZIP** button, then unzip), **or** clone it:
   ```
   git clone https://github.com/grahambrownaz/penguin-game.git
   ```
2. Double-click **`index.html`** — it opens in any modern browser (Chrome, Edge, Safari, Firefox).

That's it. `index.html` and `three.min.js` just need to stay in the same folder.

## 🎮 Controls

| Action | Keys |
| --- | --- |
| **Move** | `W` `A` `S` `D` (arrow `↑` and `→` also move) |
| **Jump** | `Space` — tap again in mid-air for one flutter-hover |
| **Attack** | `F`, `X`, or `←` — **hold** to charge a special, then release |
| **Belly-slide** | `Z` or `↓` — dash through enemies and shrug off hazards |
| **Pause** | `P` |
| **Advance menus** | `Shift` (clicks the on-screen continue button) |

> No weapon equipped? The attack key throws a quick forward **punch**.

## 🥚 The goal

- **Normal levels:** rescue every glowing **egg** (some hide inside caves, houses, enemy bases, and on sky platforms), then waddle into the **portal** to finish.
- **Boss levels (last level of each world):** defeat the **world boss**. It drops a **🗝️ key** — grab it to unlock the **castle cage** of eggs. Free those eggs, then step into the portal to move on to the next world.
- **Hidden treasure chests:** 2 per level — find them for a healing **potion** 🧪 or bonus **eggs** 🥚.

## ⚔️ Weapons (power-ups you find)

| Pickup | Tap | Hold (charged) |
| --- | --- | --- |
| 🗡️ **Sword** | quick slash (every 3rd is a heavy combo finisher) | 360° spin slash |
| 🏹 **Bow** | piercing arrow | triple shot (3 piercing arrows) |
| ❄️ **Icicle Staff** | homing icicle that freezes | 360° burst of 8 icicles |
| 🔨 **Hammer** | heavy wide swing, big knockback | ground-slam shockwave |
| 🔥 **Flame Staff** | piercing fire bolt | 5-bolt flame cone |

## 🌍 Worlds

Iceland → Desert → Jungle → Volcano → Swamp → Crystal Caverns → Storm Peaks → Shadow Realm → Clockwork Citadel — each with its own enemies, hazards, and a unique boss.

## 💡 Tips

- Eggs double as coins — spend them on penguin skins in the shop.
- Jump over enemies, or belly-slide straight through them.
- Try **Practice / Boss Practice** arenas from the menu to test weapons with unlimited ammo and no game-overs.

Have fun! 🐧
